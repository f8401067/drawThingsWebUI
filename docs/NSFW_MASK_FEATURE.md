# NSFW遮罩功能测试指南

## 功能概述

在生成结果界面，如果图片被标记为NSFW（不适合工作场所的内容），将会：
1. 对图片进行模糊遮罩处理（只能看清一点点）
2. 显示警告图标和提示文字
3. 点击遮罩后提示用户去历史记录页面开启"显示NSFW"开关才能查看原图

## 测试步骤

### 1. 准备测试环境

确保以下配置已完成：
- LLM服务已配置并运行（用于NSFW检测）
- 数据库已包含`is_nsfw`字段（运行过迁移脚本）

### 2. 测试正常图片（非NSFW）

1. 启动应用：`python app.py`
2. 访问主页：`http://localhost:5000`
3. 输入一个安全的提示词，例如："beautiful landscape with mountains"
4. 点击"生成图片"
5. 等待生成完成
6. **预期结果**：
   - 图片正常显示，无遮罩
   - 可以直接点击查看原图

### 3. 测试NSFW图片

1. 输入可能触发NSFW检测的提示词，例如："nude woman, explicit content"
2. 点击"生成图片"
3. 等待生成完成
4. **预期结果**：
   - 图片被黑色半透明遮罩覆盖（透明度85%）
   - 背景有20px模糊效果
   - 显示🔞图标
   - 显示文字："此图片包含不适合工作场所的内容"
   - 显示提示："点击查看详情"
   - 鼠标悬停时遮罩稍微变亮

5. 点击遮罩
6. **预期结果**：
   - 弹出模态框
   - 标题："查看NSFW内容"
   - 消息："此图片已被标记为不适合工作场所(NSFW)的内容。如需查看原图，请前往历史记录页面，开启"显示NSFW"开关。"
   - "历史记录"文字是可点击的链接，会在新标签页打开历史记录页面
   - 有"我知道了"按钮关闭弹窗

### 4. 测试遮罩隐藏

1. 在结果显示界面，点击"返回表单"按钮
2. **预期结果**：
   - 返回到表单界面
   - 遮罩被隐藏
   - 下次生成图片时遮罩容器重新显示

### 5. 验证历史记录联动

1. 生成一张NSFW图片
2. 点击遮罩上的"历史记录"链接
3. 在历史记录页面，确认可以看到该图片（默认隐藏）
4. 开启"显示NSFW"开关
5. **预期结果**：
   - NSFW图片显示出来
   - 可以点击放大查看原图

## 技术实现细节

### CSS样式
- `.nsfw-mask-container`: 相对定位的容器
- `.nsfw-mask`: 绝对定位的遮罩层
  - 背景：rgba(0, 0, 0, 0.85) - 85%不透明度的黑色
  - 模糊：backdrop-filter: blur(20px) - 20px高斯模糊
  - 过渡动画：0.3s ease
  - 悬停效果：背景变为75%不透明度

### JavaScript逻辑
1. `showResult()`: 显示结果时初始化遮罩容器
2. `checkNsfwStatus()`: 轮询检查图片NSFW状态（最多30秒）
3. `showNsfwMask()`: 检测到NSFW时显示遮罩
4. `showNsfwHint()`: 点击遮罩时显示提示信息
5. `backToForm()`: 返回表单时隐藏遮罩

### HTML结构
```html
<div class="nsfw-mask-container">
    <a href="..." target="_blank">
        <img src="..." alt="Generated Image">
    </a>
    <div class="nsfw-mask" onclick="showNsfwHint()">
        <div class="nsfw-mask-icon">🔞</div>
        <div class="nsfw-mask-text">此图片包含不适合工作场所的内容</div>
        <div class="nsfw-mask-hint">点击查看详情</div>
    </div>
</div>
```

## 常见问题

### Q1: 遮罩没有显示？
- 检查浏览器控制台是否有JavaScript错误
- 确认LLM配置正确，NSFW检测正在运行
- 检查数据库中`is_nsfw`字段是否被正确设置为1

### Q2: 点击图片仍然可以查看原图？
- 这是预期行为，遮罩只是视觉提示
- 用户可以通过点击遮罩了解如何正式查看NSFW内容
- 真正的访问控制需要在后端实现（可选增强）

### Q3: 遮罩样式不美观？
- 可以调整CSS中的以下参数：
  - `background: rgba(0, 0, 0, 0.85)` - 调整不透明度
  - `backdrop-filter: blur(20px)` - 调整模糊程度
  - `font-size` - 调整文字大小
